# Primary-Data-Analysis
# Data Analysis using Python

## Project Overview
This project focuses on basic **data analysis using Python**.  
The notebook demonstrates how to explore datasets, identify missing values, and derive meaningful insights using popular Python libraries.

The goal of this project is to understand data quality issues and perform initial exploratory data analysis (EDA), which is a critical step in any data science or business analytics workflow.

---

## Objectives
- Understand the structure of the dataset
- Identify and analyze missing (null) values
- Perform basic data exploration
- Generate insights from the data

---

## Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## Dataset Details
- The dataset contains multiple columns including **Income**
- Missing values are present in the dataset
- Null values are represented as:
  - `NaN`
  - `None`
  - `NaT`

---

## Key Insights
- The **Income** column contains missing values (for example, in row 10)
- Null values can impact analysis and must be handled properly
- Data cleaning is required before advanced analysis or modeling
